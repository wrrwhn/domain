客户端请求

连接

流程

  • 端通过文件事件，无阻塞地响应客户端的连接，并返回套接字描述符 fd
  • 为 fd 创建 redisClient 结构实例，并添加至客户端链表
  • 在事件处理中，为该 fd 关联文件事件
