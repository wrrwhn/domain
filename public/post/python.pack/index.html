详解

pyInstaller

注意事项

  • 依赖于打包环境，无法于 x64 环境打 x86 的包 > Pyinstaller produces a binary depending from the python you used to build it. So if you use python 2.7 64 bit it is not possible, as far as I know, to produce a 32 bit executable.

指令

  • 安装
    • pip[3] install pyinstaller
  • 版本
    • pyinstaller --version
  • 执行
    • pyinstaller [option] *.py
    • 参数
      • -F
        • 打包出可独立运行 exe
      • -c
        • 使用控制台，无界面
        • 默认项
      • -w|--noconsole
        • 去除控制台窗口，仅带 GUI 界面时推荐
        • cmd 中执行时不显示相关日志
      • -i
        • 指定可执行文件的图标
    • 示例
      • pyinstaller -F -i favicon.ico pack.py
  • 查看文件列表
    • pyi-archive_viewer *.exe
    • 示例
      • pyi-archive_viewer pack.exe pos, length, uncompressed, iscompressed, type, name [(0, 255, 323, 1, 'm', 'struct'), (255, 1104, 1814, 1, 'm', 'pyimod01_os_path'), (1359, 4287, 9268, 1, 'm', 'pyimod02_archive'), (5646, 7221, 18489, 1, 'm', 'pyimod03_importers'), (12867, 1859, 4171, 1, 's', 'pyiboot01_bootstrap'), (14726, 86, 103, 1, 's', 'pack'), (14812, 47536, 87888, 1, 'b', 'VCRUNTIME140.dll'), (62348, 39531, 87552, 1, 'b', '_bz2.pyd'), (101879, 623839, 1443840, 1, 'b', '_hashlib.pyd'), (725718, 85308, 247296, 1, 'b', '_lzma.pyd'), (811026, 28848, 65536, 1, 'b', '_socket.pyd'), (839874, 782920, 1737216, 1, 'b', '_ssl.pyd'), (1622794, 480, 1029, 1, 'b', 'pack.exe.manifest'), (1623274, 83821, 180736, 1, 'b', 'pyexpat.pyd'), (1707095, 1522436, 3555840, 1, 'b', 'python36.dll'), (3229531, 9124, 19968, 1, 'b', 'select.pyd'), (3238655, 356822, 899072, 1, 'b', 'unicodedata.pyd'), (3595477, 0, 0, 0, 'o', 'pyi-windows-manifest-filename pack.exe.manifest'), (3595477, 196507, 734166, 1, 'x', 'base_library.zip'), (3791984, 1150741, 1150741, 0, 'z', 'out00-PYZ.pyz')]
