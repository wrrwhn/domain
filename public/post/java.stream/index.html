File*Stream

  • 用法
    • 以流的形式进行文件或文件描述符的操作
      • 支持自动创建文件
      • 文件追加形式
    • 文件已打开情况下，无法进行写操作
    • 读取时推荐用于读取图片等形式数据，若读取文本，推荐使用 FileReader
