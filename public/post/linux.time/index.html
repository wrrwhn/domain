Linux.Time

时间： 2018-07-05   |   分类： 整理     |   阅读： 1304 字 ~3分钟

Command

Function

  • 测量指定指令执行所消耗的时间、系统资源

Format

  • time [options] <command [arguments...]>

result

/usr/bin/time -p ls -l 
total 2104
-rw-rw-r-- 1 appuser appuser     218 Aug 14  2017 curl-cookie-test
real 0.00
user 0.00
sys 0.00

command

  • 正常调用指定，如 ll
    • 但需要使用完整路径
      • ll 需使用 ls -l
      • 调用文件时，使用完整路径名
